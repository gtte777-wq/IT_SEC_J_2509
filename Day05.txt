반복문 : 컴퓨터 = 기계 한번의 계산으로 끝 X 반복적으로 해야될 일이 多
종류 : for 
scope : 변수, 함수, 클래스 등 식별자가 유효하게 참조될 수 있는 프로그램의 부분을 의미
 - 블록 스코프: 중괄호로 감싼 코드 블록 내 변수 유효
 - 함수 스코프: 함수 내부의 변수는 함수 내 사용 가능

함수 : (=모듈) , 기계, blackbox & whitebox
blackbox 함수 사용법이 존재한다.
function : 함수를 선언 하겠다.
함수명 : 임의로 짓되 영어로 짓는걸 권장
() : 함수 이름쪽에 주입필수
{} : () 오른쪽에 주입 필수
함수를 쓰는 궁극적인 이유 - 컴퓨터는 사칙연산과 대입연산 기반에 논리연산과 제어연산을 거쳐 작동
어려운 자료처리는 함수로 감싸고(모듈화 시키고)
